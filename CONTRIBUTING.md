# Contributing

Thanks for your interest in contributing to `gen-ai-skills`.

This repo is in early scaffolding. Conventions for skill format, directory layout, and review process will be added here as the project takes shape.

For now:

1. Fork the repository and create a feature branch from `main`.
2. Open a pull request describing the change and its motivation.
3. Be kind in code review.
